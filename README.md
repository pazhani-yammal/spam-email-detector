# Spam Email Detector 📧

A machine learning project that classifies emails/messages as spam or not spam using the Naive Bayes algorithm.

## 🛠️ Tools & Technologies
- Python
- pandas, numpy
- scikit-learn
- Google Colab

## 📊 Dataset
SMS Spam Collection Dataset — 5,574 messages labelled as ham or spam.

## ⚙️ How it works
1. Load and clean the dataset
2. Convert text to numbers using CountVectorizer
3. Train a Multinomial Naive Bayes model
4. Predict spam or ham on new messages

## 📈 Result
Achieved 98%+ accuracy on test data.
